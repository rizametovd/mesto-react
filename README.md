## Установка
Для запуска на локальной машине необходимо:</br>
1. Установить npm зависимости:</br>
```sh
npm install
```
2. Запустить в режиме разработки:</br>
```sh
npm run start
```
Если все прошло успешно, проект будет запущен на `http://localhost:3000`
